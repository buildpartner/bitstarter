FAQ

What is cloud and what is Hybrid Cloud?
Simply put: the future of computing. Here is a basic overview which explains in plain language. Cloud incorporates many areas, where we focusing is the IaaS (Infrastructure as a Service) Hybrid Cloud market specifically. Hybrid Cloud is the use of more than one public cloud provider (AWS, Rackspace, Google Compute Engine), or mixed public, private or hosted colocated environments. Hybrid cloud enables organisations (when used with the right management and orchestration interface) to dynamically and automatically migrate operating systems, applications and workloads dynamically from one provider cloud to another, usually based on cost (financial, latency, availability) and load requirements (CPU, memory, network, storage). A recent Gartner report forecast the IaaS market is expected to grow 47.3 percent in 2013 to $9 billion. There is also substantial forecast growth in this Hybrid Cloud area, particularly in enabling enterprise users on mobile devices.

What does Dashscape do?
Dashscape enables clear control of multiple cloud environments, from any device with individual saved views. Maintains simplicity through leveraging existing systems and a modular plugin approach.
Would Dashscape work with generic VPS servers or only specific Hybrid Cloud?
Yes, Dashscape will work with your VPS servers and have cPanel and Plesk control panel support. Initially support will be for OpenStack Horizon and Rackspace control panel, then for other public cloud providers such as AWS and GCE management consoles. The main Dashscape feature Plugin available from the outset will be to Ansible, enabling true management, monitoring and orchestration from a single pane via SSH to any capable system without the need for agent software. Typical orchestration requires agents installed on the target system during initial system build and is therefore complicated. These current Hybrid Cloud managment solutions offer a terrible and unnecessarily complicated user interface. With Dashscape you will be able to manage across multiple provider and Hybrid Cloud environments, simply and elegantly.
